안녕하세요, 저는 웹 디자이너입니다.

사용자 경험을 최우선으로 생각하며 아름다운 웹사이트를 만듭니다.

디자인 컨셉

본 포트폴리오 웹사이트는 모던하고 미니멀한 디자인을 추구합니다. 최신 웹 디자인 트렌드인 Glassmorphism과 Bento Grid 레이아웃을 활용하여 시각적인 매력과 사용성을 동시에 확보합니다.

색상 팔레트

#007bff
#28a745
#dc3545
#f8f9fa
#343a40

**주요 색상:**

  • **#007bff (Blue):** 주요 액션 버튼 및 링크에 사용되어 사용자의 시선을 유도합니다.
  • **#28a745 (Green):** 긍정적인 피드백 또는 성공적인 결과에 사용됩니다.
  • **#dc3545 (Red):** 오류 또는 경고 메시지에 사용됩니다.
  • **#f8f9fa (Light Gray):** 배경색으로 사용하여 콘텐츠를 돋보이게 합니다.
  • **#343a40 (Dark Gray):** 텍스트 및 헤딩에 사용되어 가독성을 높입니다.

폰트 스타일

제목

Noto Sans KR - Bold

본문

Noto Sans KR - Regular

**폰트:** Noto Sans KR (Google Fonts)

  • 제목: Bold (700)
  • 본문: Regular (400)
Noto Sans KR은 가독성이 뛰어나고 다양한 언어를 지원하는 폰트로, 웹사이트의 전체적인 통일성을 유지하면서도 정보를 명확하게 전달하는 데 효과적입니다.

레이아웃

주요 레이아웃 요소

  • **헤더:** 웹사이트 제목, 네비게이션 메뉴를 포함합니다. 페이지 상단에 고정되어 사용자가 언제든지 다른 섹션으로 이동할 수 있도록 합니다.
  • **Hero 섹션:** 웹사이트의 첫인상을 결정짓는 중요한 영역입니다. 개인의 핵심 역량과 가치를 간결하게 소개합니다.
  • **소개 (About) 섹션:** 개인의 배경, 경험, 기술, 가치관 등을 상세하게 설명하여 신뢰도를 높입니다.
  • **포트폴리오 (Portfolio) 섹션:** 개인의 기술과 능력을 입증하는 핵심 섹션으로, 최고의 프로젝트를 선별하여 상세하게 제시합니다. Bento Grid 레이아웃을 사용하여 시각적인 다양성을 제공합니다.
  • **연락처 (Contact) 섹션:** 잠재적 고객 또는 고용주가 쉽게 연락할 수 있도록 명확하고 간편한 연락 방법을 제공합니다.
  • **푸터:** 저작권 정보, 소셜 미디어 링크 등을 포함합니다.

UI 컴포넌트

웹사이트의 UI 컴포넌트는 사용자의 편의성을 고려하여 디자인되었습니다. 주요 컴포넌트는 다음과 같습니다.

  • **버튼:** 주요 액션 (예: 자세히 보기, 문의하기)을 유도하는 데 사용됩니다.
  • **입력 양식:** 사용자로부터 정보를 수집하는 데 사용됩니다.
  • **이미지:** 프로젝트 결과물 또는 개인 프로필 사진을 보여주는 데 사용됩니다.
  • **아이콘:** 시각적인 단서를 제공하고 사용자 경험을 향상시키는 데 사용됩니다.
